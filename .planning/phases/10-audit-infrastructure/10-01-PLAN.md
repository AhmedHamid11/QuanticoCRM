Create migration 049_create_audit_logs.sql:
- Table: audit_logs with columns:
  - id TEXT PRIMARY KEY
  - org_id TEXT NOT NULL (FK to organizations)
  - event_type TEXT NOT NULL
  - actor_id TEXT (nullable - some events have no actor)
  - actor_email TEXT
  - target_id TEXT (nullable)
  - target_type TEXT (nullable)
  - ip_address TEXT
  - user_agent TEXT
  - details TEXT (JSON blob for event-specific data)
  - success INTEGER NOT NULL DEFAULT 1
  - error_msg TEXT
  - prev_hash TEXT NOT NULL (previous entry's hash, "GENESIS" for first)
  - entry_hash TEXT NOT NULL UNIQUE (this entry's hash)
  - created_at TEXT NOT NULL DEFAULT CURRENT_TIMESTAMP
- Indexes:
  - idx_audit_logs_org_created (org_id, created_at DESC) - for timeline queries
  - idx_audit_logs_org_type (org_id, event_type) - for filtering
  - idx_audit_logs_actor (org_id, actor_id) - for user filtering
  - idx_audit_logs_prev_hash (prev_hash) - for chain verification
- NO foreign key on actor_id/target_id (audit logs must persist even if user deleted)

- Add AuditLogEntry struct with all DB fields + json/db tags

  - Concatenate: ID|OrgID|EventType|ActorID|ActorEmail|TargetID|TargetType|IPAddress|Details|Success|ErrorMsg|PrevHash|CreatedAt(RFC3339Nano)
  - Return hex-encoded SHA-256 hash
- Keep existing AuditEvent struct for backwards compatibility (it's the input)

Implement methods:

1. GetLastEntryHash(ctx, orgID string) (string, error)
   - Query: SELECT entry_hash FROM audit_logs WHERE org_id = ? ORDER BY created_at DESC LIMIT 1
   - Return "GENESIS" if sql.ErrNoRows (first entry in org's chain)

2. Create(ctx, entry *entity.AuditLogEntry) error
   - Generate ID using util.NewSFID("aud")
   - Get prev_hash via GetLastEntryHash
   - Set entry.PrevHash = prevHash
   - Compute entry.EntryHash = entry.ComputeEntryHash()
   - INSERT into audit_logs
   - Return error if any step fails

3. List(ctx, orgID string, filters *AuditLogFilters) (*AuditLogListResponse, error)
   - AuditLogFilters: EventTypes []string, ActorID string, DateFrom/DateTo *time.Time, Page int, PageSize int
   - Build query with WHERE clauses based on filters
   - Support pagination (LIMIT/OFFSET)
   - Return items, total count, hasMore flag
   - Order by created_at DESC (newest first)

4. VerifyChainIntegrity(ctx, orgID string) (*ChainVerificationResult, error)
   - Query all entries for org ordered by created_at ASC
   - Verify each entry's prev_hash matches previous entry's entry_hash
   - Verify each entry's entry_hash is correctly computed
   - Return result with: Valid bool, EntriesVerified int, Errors []string
  </action>
