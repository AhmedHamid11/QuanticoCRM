    -- Table: api_usage_log (tenant DB) - tracks API calls per org over rolling 24-hour windows

        timestamp TEXT NOT NULL,

  <action>
    Create `backend/internal/service/salesforce_ratelimit.go` with the following:

    1. **RateLimitService struct:**
    ```go
    type RateLimitService struct {
        dbManager *db.Manager
        authRepo  *repo.AuthRepo
    }
    ```
    Constructor: `NewRateLimitService(dbManager *db.Manager, authRepo *repo.AuthRepo) *RateLimitService`

    The service uses tenant DB for all queries (API usage is per-tenant). Follow the same pattern as salesforce_delivery.go for getting tenant DB: call `authRepo.GetOrganizationByID` then `dbManager.GetTenantDB`.

    2. **GetAPIUsageLast24Hours(ctx, orgID string) (int, error):**
    - Calculate cutoff: `time.Now().UTC().Add(-24 * time.Hour).Format(time.RFC3339)`
    - Query: `SELECT COALESCE(SUM(api_calls), 0) FROM api_usage_log WHERE org_id = ? AND timestamp >= ?`
    - Handle "no such table" gracefully (return 0, nil) since tenant DBs may not have migration yet
    - Returns total API calls in last 24 hours

    3. **CanMakeAPICalls(ctx, orgID string, callCount int) (bool, error):**
    - Calls GetAPIUsageLast24Hours to get current usage
    - Returns true if `usage + callCount <= entity.SalesforcePauseThreshold` (80K)
    - Returns false if at or above threshold

    4. **RecordAPIUsage(ctx, orgID, jobID string, callCount int) error:**
    - Insert into api_usage_log with sfid.New("apilog") for ID
    - timestamp = `time.Now().UTC().Format(time.RFC3339)`
    - Handle "no such table" gracefully (log warning, return nil) - non-critical operation
    - Also update sync_jobs.api_calls_made for the job: `UPDATE sync_jobs SET api_calls_made = ? WHERE id = ?`

    5. **GetQuotaStatus(ctx, orgID string) (*entity.QuotaStatus, error):**
    - Calls GetAPIUsageLast24Hours
    - Calculates percentage: `(usage * 100) / entity.SalesforceMaxDailyAPICalls`
    - Returns QuotaStatus with usage, limit (100000), percentage, threshold (80000), isPaused (usage >= threshold)

    6. **CleanupOldUsage(ctx, orgID string) error:**
    - Delete records older than 25 hours: `DELETE FROM api_usage_log WHERE org_id = ? AND timestamp < ?`
    - 25 hours (not 24) provides buffer to avoid deleting records right at the boundary
    - Handle "no such table" gracefully

    IMPORTANT: All tenant DB access must follow the pattern:
    ```go
    org, err := s.authRepo.GetOrganizationByID(ctx, orgID)
    // handle error
    tenantDB, err := s.dbManager.GetTenantDB(ctx, orgID, org.DatabaseURL, org.DatabaseToken)
    // handle error, fallback to master DB if no DatabaseURL
    ```

    Use `strings.Contains(err.Error(), "no such table")` pattern for graceful table-missing handling (same as ListJobs in salesforce_delivery.go).
  </action>
