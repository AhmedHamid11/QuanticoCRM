**StreamProgress (SSE endpoint):**
```go
func (h *ScanJobHandler) StreamProgress(c *fiber.Ctx) error {
    orgID := c.Locals("orgID").(string)
    subscriberID := uuid.New().String() // or use simple counter

    c.Set("Content-Type", "text/event-stream")
    c.Set("Cache-Control", "no-cache")
    c.Set("Connection", "keep-alive")
    c.Set("Transfer-Encoding", "chunked")

    c.Status(fiber.StatusOK).Context().SetBodyStreamWriter(
        fasthttp.StreamWriter(func(w *bufio.Writer) {
            eventChan := h.subscribe(orgID, subscriberID)
            defer h.unsubscribe(orgID, subscriberID)

            // Send initial heartbeat
            fmt.Fprintf(w, ": heartbeat\n\n")
            w.Flush()

            for {
                select {
                case event := <-eventChan:
                    data, _ := json.Marshal(event)
                    fmt.Fprintf(w, "event: progress\ndata: %s\n\n", data)
                    if err := w.Flush(); err != nil {
                        return // Client disconnected
                    }
                    if event.Status == "completed" || event.Status == "failed" {
                        // Don't close — other jobs may still be running
                        // Client can close when no more active jobs
                    }
                case <-time.After(30 * time.Second):
                    // Keepalive ping every 30 seconds
                    fmt.Fprintf(w, ": keepalive\n\n")
                    if err := w.Flush(); err != nil {
                        return
                    }
                case <-c.Context().Done():
                    return
                }
            }
        }),
    )
    return nil
}
```

**subscribe/unsubscribe helpers:** Manage per-org subscriber channels. Channel buffer size 20.

**broadcastProgress(event ProgressEvent):** Read-lock subMu, iterate subscribers for event.OrgID, send event to each channel (non-blocking with select/default to skip full channels).

**ListNotifications:** Get orgID + userID from locals. Parse query: includeRead (bool), page, pageSize. Call notificationRepo.ListForUser. Return paginated response.

**UnreadCount:** Get orgID + userID. Call notificationRepo.CountUnread. Return `{count: N}`.

**MarkAsRead:** Get notification ID. Call notificationRepo.MarkAsRead.

**MarkAllAsRead:** Get orgID + userID. Call notificationRepo.MarkAllAsRead.

**DismissNotification:** Get notification ID. Call notificationRepo.Dismiss.

IMPORTANT:
