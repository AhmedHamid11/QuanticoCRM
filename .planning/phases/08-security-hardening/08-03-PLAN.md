<task type="auto">
  <name>Task 1: Create PasswordInput component with strength meter</name>
  <files>fastcrm/frontend/src/lib/components/PasswordInput.svelte</files>
  <action>
Create a reusable password input component with real-time strength feedback.

The component should:
1. Accept props: `value` (bindable), `id`, `name`, `placeholder`, `error`, `autocomplete`
2. Show password strength indicator (4 levels: Too weak, Weak, Medium, Strong)
3. Show character count hint when under 8 characters
4. Use Tailwind for styling to match existing components

Password strength calculation (simple, no external library):
- Too weak (red): length < 8
- Weak (orange): length >= 8, only letters or only numbers
- Medium (yellow): length >= 8, has letters AND numbers
- Strong (green): length >= 12 AND has letters, numbers, and special chars

```svelte
<script lang="ts">
    let {
        value = $bindable(''),
        id = 'password',
        name = 'password',
        placeholder = 'Enter password',
        error = '',
        autocomplete = 'new-password'
    } = $props();

    // Calculate strength
    let strength = $derived(() => {
        if (!value) return null;
        const len = value.length;
        if (len < 8) return { level: 'Too weak', color: 'bg-red-500', width: 'w-1/4' };

        const hasLetters = /[a-zA-Z]/.test(value);
        const hasNumbers = /[0-9]/.test(value);
        const hasSpecial = /[^a-zA-Z0-9]/.test(value);

        if (len >= 12 && hasLetters && hasNumbers && hasSpecial) {
            return { level: 'Strong', color: 'bg-green-500', width: 'w-full' };
        }
        if (hasLetters && hasNumbers) {
            return { level: 'Medium', color: 'bg-yellow-500', width: 'w-3/4' };
        }
        return { level: 'Weak', color: 'bg-orange-500', width: 'w-2/4' };
    });
</script>

<div class="space-y-2">
    <input
        type="password"
        {id}
        {name}
        {autocomplete}
        required
        bind:value
        class="appearance-none relative block w-full px-3 py-2 border border-gray-300 rounded-md placeholder-gray-400 text-gray-900 focus:outline-none focus:ring-blue-500 focus:border-blue-500 sm:text-sm"
        class:border-red-500={error}
        {placeholder}
    />

    {#if value}
        <div class="space-y-1">
            <div class="h-1.5 bg-gray-200 rounded-full overflow-hidden">
                <div
                    class="h-full transition-all duration-300 {strength()?.color} {strength()?.width}"
                ></div>
            </div>
            <div class="flex justify-between text-xs">
                <span class="text-gray-500">
                    {#if value.length < 8}
                        {8 - value.length} more characters needed
                    {:else}
                        Strength: {strength()?.level}
                    {/if}
                </span>
                <span class="text-gray-400">{value.length} characters</span>
            </div>
        </div>
    {/if}

    {#if error}
        <p class="text-sm text-red-600">{error}</p>
    {/if}
</div>
```
