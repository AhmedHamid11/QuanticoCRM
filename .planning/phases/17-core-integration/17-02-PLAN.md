Create service/salesforce_oauth.go with `SalesforceOAuthService`:

**Constructor:**
```go
type SalesforceOAuthService struct {
    repo          *repo.SalesforceRepo
    encryptionKey []byte
}

func NewSalesforceOAuthService(repo *repo.SalesforceRepo, encryptionKey []byte) *SalesforceOAuthService
```

**Methods:**

1. `SaveConfig(ctx, orgID string, config entity.SFSyncConfig) error`:
   - Encrypt client_secret using util.EncryptToken
   - Create SalesforceConnection entity with ClientID, encrypted secret, RedirectURL
   - Call repo.UpsertConnection
   - Return any errors with clear messages

2. `GetConfig(ctx, orgID string) (*entity.SalesforceConnection, error)`:
   - Call repo.GetConnection
   - Return connection (tokens remain encrypted in returned struct)
   - Return nil, nil if no connection exists (not an error)

3. `GetAuthorizationURL(ctx, orgID string) (string, error)`:
   - Load connection from repo to get ClientID, ClientSecret, RedirectURL
   - Decrypt client_secret
   - Build oauth2.Config with:
    - ClientID, ClientSecret from connection
    - Endpoint: AuthURL from env `SALESFORCE_AUTH_URL` (default: `https://login.salesforce.com/services/oauth2/authorize`), TokenURL from env `SALESFORCE_TOKEN_URL` (default: `https://login.salesforce.com/services/oauth2/token`)
    - RedirectURL from connection
    - Scopes: ["api", "refresh_token"]
   - Generate CSRF state token: base64(orgID + ":" + random32bytes)
   - Return oauth2Config.AuthCodeURL(state, oauth2.AccessTypeOffline)

4. `HandleCallback(ctx, code, state string) (string, error)`:
   - Parse state to extract orgID (base64 decode, split on ":")
   - Load connection from repo
   - Decrypt client_secret to build oauth2.Config
   - Exchange code for token: oauth2Config.Exchange(ctx, code)
   - Extract instance_url from token.Extra("instance_url")
   - Encrypt access_token and refresh_token using util.EncryptToken
   - Store encrypted tokens via repo.UpdateTokens
   - Update connection's instance_url and connected_at
   - Return orgID for redirect

5. `GetHTTPClient(ctx, orgID string) (*http.Client, error)`:
   - Load connection from repo
   - Decrypt access_token and refresh_token
   - Build oauth2.Token from decrypted values
   - Build oauth2.Config (same pattern as GetAuthorizationURL)
   - Return oauth2Config.Client(ctx, token) - this auto-refreshes expired tokens
   - IMPORTANT: After client creation, check if token was refreshed (compare expiry). If refreshed, re-encrypt and store new tokens via repo.UpdateTokens
   - This is the proactive refresh mechanism: golang.org/x/oauth2 refreshes automatically when token is expired or close to expiry

6. `GetConnectionStatus(ctx, orgID string) (string, error)`:
   - Load connection from repo
   - If no connection: return "not_configured"
   - If no access_token: return "configured" (credentials saved but not yet authorized)
   - Decrypt access_token, check expires_at
   - If expired and no refresh_token: return "expired"
   - If connected: return "connected"

7. `DisconnectOrg(ctx, orgID string) error`:
   - Clear tokens from connection (set access_token_encrypted, refresh_token_encrypted to nil)
   - Set connected_at to nil
   - Repo update

Environment variable handling:
- `SALESFORCE_AUTH_URL` (default: https://login.salesforce.com/services/oauth2/authorize)
- `SALESFORCE_TOKEN_URL` (default: https://login.salesforce.com/services/oauth2/token)
- Encryption key already in util.GetEncryptionKey() but pass via constructor for testability

SECURITY NOTES (from PITFALLS.md):
- Never log tokens (only log orgID and operation name)
- Validate state parameter on callback to prevent CSRF
- Use oauth2.AccessTypeOffline to get refresh_token
- Support sandbox URLs via environment variables (don't hardcode login.salesforce.com)

Create handler/salesforce.go with `SalesforceHandler`:

**Constructor:**
```go
type SalesforceHandler struct {
    oauthService *service.SalesforceOAuthService
    repo         *repo.SalesforceRepo
}

func NewSalesforceHandler(oauthService *service.SalesforceOAuthService, repo *repo.SalesforceRepo) *SalesforceHandler
```

**Endpoints:**

1. `POST /salesforce/config` - SaveConfig:
   - Parse SFSyncConfig from body (clientId, clientSecret, redirectUrl)
   - Validate: all three fields required, clientId non-empty, clientSecret non-empty
   - Call oauthService.SaveConfig(ctx, orgID, config)
   - Return 200 with {"status": "configured"}

2. `GET /salesforce/config` - GetConfig:
   - Call oauthService.GetConfig(ctx, orgID)
   - Return config but NEVER return encrypted tokens or client_secret
   - Return: {clientId, redirectUrl, instanceUrl, isEnabled, status (from GetConnectionStatus), connectedAt}

3. `POST /salesforce/oauth/authorize` - InitiateOAuth:
   - Call oauthService.GetAuthorizationURL(ctx, orgID)
   - Return 200 with {"authUrl": url}

4. `GET /salesforce/oauth/callback` - OAuthCallback:
   - Extract query params: code, state
   - Call oauthService.HandleCallback(ctx, code, state)
   - Redirect to frontend: `/admin/integrations/salesforce?status=connected`
   - On error: redirect with `?status=error&message=...`

5. `GET /salesforce/status` - GetStatus:
   - Call oauthService.GetConnectionStatus(ctx, orgID)
   - Return 200 with {"status": "connected"|"configured"|"not_configured"|"expired"}

6. `POST /salesforce/disconnect` - Disconnect:
   - Call oauthService.DisconnectOrg(ctx, orgID)
   - Return 200 with {"status": "disconnected"}

7. `PUT /salesforce/toggle` - ToggleSync:
   - Parse body: {"enabled": bool}
   - Call repo.SetEnabled(ctx, orgID, enabled)
   - Return 200 with {"isEnabled": enabled}

**RegisterRoutes method:**
```go
func (h *SalesforceHandler) RegisterRoutes(router fiber.Router) {
    sf := router.Group("/salesforce")
    sf.Post("/config", h.SaveConfig)
    sf.Get("/config", h.GetConfig)
    sf.Post("/oauth/authorize", h.InitiateOAuth)
    sf.Get("/oauth/callback", h.OAuthCallback)
    sf.Get("/status", h.GetStatus)
    sf.Post("/disconnect", h.Disconnect)
    sf.Put("/toggle", h.ToggleSync)
}
```
