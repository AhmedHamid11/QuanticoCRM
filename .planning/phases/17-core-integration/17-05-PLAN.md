**Page structure** (3 sections):

- Card with title "Salesforce Connected App"
- Form fields (Svelte 5 $state):
  - Client ID (text input, required)
  - Client Secret (password input, required)
  - Redirect URL (text input, required, pre-fill with `{window.location.origin}/api/v1/salesforce/oauth/callback`)
- "Save Configuration" button -> POST /api/v1/salesforce/config
- Show success toast on save

**Section 2: Connection Status & OAuth**
- Card with title "Connection Status"
- Status indicator with colored dot:
  - Green dot + "Connected" + connected date
  - Yellow dot + "Configured" + "Click Connect to authorize"
  - Red dot + "Expired" + "Token expired, reconnect required"
  - Gray dot + "Not Configured" + "Enter Connected App details above"
- "Connect to Salesforce" button (primary, blue) -> POST /api/v1/salesforce/oauth/authorize, then window.location.href = authUrl
- "Disconnect" button (danger, red) -> POST /api/v1/salesforce/disconnect
- Toggle switch for "Enable Sync" -> PUT /api/v1/salesforce/toggle
- Load status on mount: GET /api/v1/salesforce/status and GET /api/v1/salesforce/config

**Section 3: Delivery Management**
- Card with title "Merge Instruction Delivery"
- "Trigger Delivery" button (only visible when connected) -> POST /api/v1/salesforce/trigger with `{"all": true}`
- Recent sync jobs table (load from GET /api/v1/salesforce/jobs?limit=10):
  - Columns: Batch ID, Entity Type, Status (badge), Instructions (delivered/total), Created, Actions
  - Status badges: green for completed, yellow for pending/running, red for failed
  - "Retry" button on failed jobs -> POST /api/v1/salesforce/jobs/{id}/retry

**Handle OAuth callback redirect:**
On mount, check URL query params:
```javascript
const params = new URLSearchParams(window.location.search);
if (params.get('status') === 'connected') {
    toast.success('Successfully connected to Salesforce');
    // Clean URL
    goto('/admin/integrations/salesforce', { replaceState: true });
}
if (params.get('status') === 'error') {
    toast.error(params.get('message') || 'Failed to connect to Salesforce');
    goto('/admin/integrations/salesforce', { replaceState: true });
}
```
